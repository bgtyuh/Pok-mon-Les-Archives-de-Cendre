# Cendrebourg — Épilogue du Sceau

- **Mère** : « Tu dors mieux. Moi aussi un peu. Mais j'ai parfois peur qu'on referme encore trop vite ce qu'on vient juste d'ouvrir. »
- **Boulangère** : « Les livraisons sont régulières, les gens plus calmes. J'essaie de croire que c'est une vraie paix. »
- **Facteur rural** : « Le centre répond plus vite aux demandes. Il pose aussi plus de cases à cocher. »
- **Jeune dresseuse** : « On nous dit que le monde est stabilisé. J'espère juste qu'on aura encore le droit de poser des questions. »
