# Scene flags

Ce document convertit les grandes scènes déjà écrites en blocs d'implémentation ROM hack. Les noms de variables proposés ci-dessous servent de convention stable pour le scripting ; ils sont déduits des scènes présentes dans `04_scenario/structure-complete.md` et `09_script-integral/`.

## Convention proposée
- `SCN_*` : scène vue ou résolue.
- `STAGE_*` : progression de zone ou d'acte.
- `REL_ADMIN`, `REL_DISS`, `LISTEN`, `REPAIR`, `ZYGARDE` : variables systémiques principales.
- `ROUTE_FINAL` : `0` indéterminée, `1` Sceau, `2` Flamme, `3` Alliance.
- `JOURNAL_*` : entrée du Carnet d'Archive déverrouillée.

## Notes de mise en œuvre
- Les scènes ci-dessus sont celles qui portent soit la progression dramatique, soit une conséquence systémique claire ; les combats de dresseurs mineurs et dialogues d'ambiance peuvent rester pilotés par des flags locaux de map.
- Quand une scène n'altère pas encore explicitement une variable dans les documents source, la modification proposée ici sert de base d'implémentation pour harmoniser la route Alliance et les variations du rival.
- Les entrées du Carnet peuvent être stockées comme flags booléens indépendants, puis regroupées par page thématique côté UI.
